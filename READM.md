# Atividade - Módulo de Front-End Estático

## Objetivo

Desenvolver uma página web utilizando HTML & CSS ou o Bootstrap 5, a página pode representar:

### 1 - Site de Notícias:  Vocês devem criar pelo menos 3 páginas. Uma página principal e duas páginas que detalham duas notícias. O tema do site pode ser sobre esportes, tecnologia, séries, filmes ou site de notícias em geral.

Requisitos:

A página deve ser responsiva.<br>
Deve ser desenvolvida utilizando HTML semântico `<header>`, `<nav>`, `<main>`, `<article>` e `<footer>`.<br>
A navegação entre as páginas deve funcionar.<br>
A página deve conter imagens e formulários.

Referências:

1 - Tecmundo: https://www.tecmundo.com.br/<br>
2 - Ge.globo: https://ge.globo.com/

### 2 - Ecommerce: Vocês devem criar um ecommerce de produtos ou de alimentos. A página inicial deve conter diversos produtos com fotos, nomes e preço. Além disso, façam duas páginas que detalham os produtos para tornar a página mais completa. 

Requisitos:
